# Supply-Chain-Delivery-Performance-Analysis
Supply-Chain-Delivery-Performance-Analysis 

# Problem Statement 
This project analyzes delivery performance for a logistics company facing issues with delays and customer dissatisfaction. The objective is to identify key bottlenecks, understand root causes, and provide actionable insights to improve delivery efficiency.

# Dashboard Overview
This dashboard provides:
1. KPI metrics (On-Time %, Delay %, Avg Delivery Days, Total Orders, Review Score)
2. Delivery performance trends over time
3. Delay analysis by city, product category, and seller
4. Relationship between delivery time and customer ratings

# Key Insights
1. Delay rate increased from 5.40% (2017) to 7.35% (2018)
2. Major delays concentrated in São Paulo and Rio de Janeiro
3. Categories like beleza_saude and cama_mesa_banho drive most delays
4. Few sellers contribute disproportionately to delays
5. Customer ratings drop significantly after 30–40 days delivery time

# Business Recommendations
1. Improve logistics capacity in high-demand cities
2. Monitor and optimize underperforming sellers
3. Improve handling of delay-prone product categories
4. Reduce delivery time to improve customer satisfaction
