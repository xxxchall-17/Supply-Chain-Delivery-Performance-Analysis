# Supply Chain & Delivery Performance Intelligence Dashboard
A dynamic, interactive Power BI dashboard designed to analyze delivery efficiency, identify delay root causes, and evaluate customer experience across multiple dimensions such as city, seller, and product category.

# Purpose
The Delivery Performance Intelligence Dashboard is an interactive Power BI solution built to analyze logistics performance and uncover key factors contributing to delayed deliveries. It helps stakeholders identify bottlenecks, understand customer impact, and take data-driven actions to improve operational efficiency and service quality.

#Tech Stack
The dashboard was built using the following tools and technologies:
📊 Power BI Desktop – Primary tool for data visualization and dashboard development
📂 Power Query – Data cleaning, transformation, and preprocessing
🧠 DAX (Data Analysis Expressions) – Used for KPIs, calculated measures, and business logic
🔗 Data Modeling – Relationship building across multiple datasets (orders, customers, sellers, reviews)
📁 File Format – .pbix for development and .png for dashboard previews

# Data Source
# Source: Brazilian E-commerce Public Dataset (Olist)
The dataset includes multiple interconnected tables such as:
● Orders data (purchase, delivery, status)
● Order items (including freight cost)
● Customers and sellers information
● Customer reviews and ratings

# Key Insights
1. Delay rate increased from 5.40% (2017) to 7.35% (2018)
2. Major delays concentrated in São Paulo and Rio de Janeiro
3. Categories like beleza_saude and cama_mesa_banho drive most delays
4. Few sellers contribute disproportionately to delays
5. Customer ratings drop significantly after 30–40 days delivery time

# Business Recommendations
1. Improve logistics capacity in high-demand cities
2. Monitor and optimize underperforming sellers
3. Improve handling of delay-prone product categories
4. Reduce delivery time to improve customer satisfaction

# Tools & Technologies
The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main data visualization platform used for report creation.
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.

# Features / Highlights

# • Business Problem
A logistics company is facing challenges related to:
• Increasing delivery delays
• Declining customer satisfaction
• Operational inefficiencies in specific regions and categories

# • Goal of the Dashboard
To develop an interactive analytics solution that:
• Tracks delivery performance using key KPIs
• Identifies delay patterns across cities, sellers, and product categories
• Analyzes the relationship between delivery time and customer ratings
• Provides actionable insights to improve logistics efficiency

# • Business Impact & Insights
• Delay rate increased from 5.40% (2017) to 7.35% (2018), indicating performance decline with scale
• High-demand cities like São Paulo and Rio de Janeiro are major delay hotspots
• Specific product categories contribute disproportionately to delays
• A small group of sellers drives a large share of delivery issues
• Customer ratings drop significantly when delivery exceeds 30–40 days, directly impacting satisfaction

# • Business Recommendations
• Improve logistics infrastructure in high-demand cities
• Monitor and optimize underperforming sellers
• Enhance handling and delivery processes for delay-prone categories
• Scale operations in line with increasing order demand
